# FaraClaudeCreateRepository

AIが自動でプロジェクトを作成するシステム

## 🚀 使い方

1. [Issues](../../issues)タブを開く
2. 「New issue」をクリック
3. 「新しいプロジェクトをリクエスト」を選択
4. フォームに記入して送信
5. 数分後、新しいリポジトリが自動作成されます！

## 📋 必要な設定

- `ANTHROPIC_API_KEY`: Claude APIキー
- `PAT_TOKEN`: GitHub Personal Access Token

## ⚡ 生成できるプロジェクト例

- ToDoアプリ
- ブログサイト
- ランディングページ
- ライブラリ/パッケージ
- その他、あらゆるプロジェクト
